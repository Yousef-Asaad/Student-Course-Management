# Student Course Management System

This is a **C++ console application** for managing students and their enrolled courses.

## Features
- Add, remove, search, and display students
- Enroll students in courses
- Show courses of a student
- Sort students by GPA
- Data is saved to `students.txt`

## How to Run
1. Open `main.cpp` in Visual Studio
2. Build and run the program
3. Use the menu to manage students and courses

## Notes
- Each student must have a unique ID
- GPA must be between 0.0 and 4.0
- Duplicate courses are not allowed
